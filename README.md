A fun web-based quiz game where the user predicts answers by flipping a coin! Each correct answer increases your coins and streak, and you can see how lucky you are.

## Features

- Flip a virtual coin to predict your answer (A or B).

- Tracks score and streak for correct answers.

- Highlights the selected option after a coin flip.

- Shows a result page at the end of the quiz.
